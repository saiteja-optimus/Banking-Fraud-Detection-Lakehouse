# Banking-Fraud-Detection-Lakehouse
Banking Fraud Detection Lakehouse is an enterprise-grade data engineering project that builds a scalable fraud analytics platform using Databricks. It ingests high-volume banking data through Auto Loader and streaming pipelines, leveraging Medallion Architecture, Lakeflow Declarative Pipelines, and Delta Lake for reliable, real-time data processing
Live site url: https://banking-fraud-detection-lakehouse.vercel.app/
